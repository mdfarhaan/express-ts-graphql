# Template for Express-Typescript with GraphQL Project

## Installation

Install dependencies

```
$ npm install
```

Build Server

```
$ npm build
```

Run Server

```
$ npm start
```

## Run using Docker🐋

Build Image
```
docker build -t <name> .
```

Run Container
```
docker run --env-file ./.env -p 5000:5000 <name>
```
